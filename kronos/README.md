# 🚀 MindFlow AI - Telegram Bot с монетизацией

AI-ассистент для планирования задач с геймификацией и монетизацией через Telegram Stars.

## ✨ Возможности

- 📝 **Управление задачами** - создание, редактирование, выполнение
- 🤖 **AI-парсинг** - понимает задачи на естественном языке
- 📊 **Планирование дня** - автоматическое составление расписания
- 🎮 **Геймификация** - уровни, XP, достижения, серии
- 💰 **Монетизация** - Premium через Telegram Stars
- 📱 **WebApp** - красивый интерфейс для Telegram

## 🛠 Установка

### 1. Клонирование

```bash
cd C:\kronos
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка окружения

Создайте файл `.env`:

```env
BOT_TOKEN=ваш_токен_от_botfather
OPENAI_API_KEY=ваш_openai_ключ_или_пусто
WEBAPP_URL=https://ваш-домен.com/webapp
API_URL=https://ваш-домен.com/api
ADMIN_IDS=ваш_telegram_id
PROXY_URL=http://user:pass@host:port
```

### 4. Запуск

```bash
python main.py
```

## 📱 Настройка Telegram Stars

1. Откройте @BotFather
2. Выберите бота → `/newapp` → подтвердите
3. Подключите платежи Stars
4. Готово! Пользователи смогут покупать Premium
